# Nestjs + Deta.sh micro starter 


## Description

Rapid API prototyping with nestjs and Deta.sh

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

[Deta.sh](https://www.deta.sh/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
